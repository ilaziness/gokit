# gokit
components

目录结构：

```shell
pkg                   // 公共包
    ├─base              // 基础包
    │  ├─constant       // 常量定义
    │  ├─errcode        // 错误码
    │  └─reqp       // 返回响应方法
    ├─config            // 配置
    ├─log               // 日志
    ├─middleware        // 通用中间件
    ├─queue             // 队列
    │  └─rocketmq       // rocket mq
    ├─storage           // 存储
    ├─timer             // 定时器
    ├─utils             // 工具方法
    └─webapp            // 应用启动入口
```
